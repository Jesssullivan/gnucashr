# Consolidation Summary Report

Generate a summary of the consolidation process.

## Usage

``` r
consolidation_summary(books, ic_rules = list(), as_of = Sys.Date())
```

## Arguments

- books:

  List of GnuCashDB objects

- ic_rules:

  Intercompany rules

- as_of:

  Date for balances

## Value

List with consolidation details
