# Monthly Activity Summary

Summarize account activity by month.

## Usage

``` r
monthly_activity(gc, account = NULL, start = NULL, end = NULL)
```

## Arguments

- gc:

  GnuCashDB object

- account:

  Account name or path (NULL for all accounts)

- start:

  Start date

- end:

  End date

## Value

tibble with monthly summaries
