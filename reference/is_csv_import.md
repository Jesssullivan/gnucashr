# Check if Object is a gnucashr CSV Import

Check if Object is a gnucashr CSV Import

## Usage

``` r
is_csv_import(x)
```

## Arguments

- x:

  Object to check

## Value

Logical TRUE if x is a gnucashr_csv_import

## Examples

``` r
if (FALSE) { # \dontrun{
import <- import_paypal_csv("paypal_export.csv")
is_csv_import(import)  # TRUE
} # }
```
