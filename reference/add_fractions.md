# Add Two Fractions

Vectorized fraction addition maintaining exact representation. Used for
accumulating balances without floating-point drift.

## Usage

``` r
add_fractions(num1, denom1, num2, denom2)
```

## Arguments

- num1:

  First numerator vector

- denom1:

  First denominator vector

- num2:

  Second numerator vector

- denom2:

  Second denominator vector

## Value

IntegerMatrix with columns: numerator, denominator
