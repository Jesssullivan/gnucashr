# Format Currency for Display

Helper function to format currency values for display.

## Usage

``` r
format_currency(x, prefix = "$", suffix = "", digits = 2)
```

## Arguments

- x:

  Numeric value

- prefix:

  Currency prefix (default "\$")

- suffix:

  Currency suffix (default "")

- digits:

  Decimal digits (default 2)

## Value

Formatted character string
